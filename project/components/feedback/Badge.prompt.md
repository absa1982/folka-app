Small soft-tinted label for counts, categories and metadata (department, plan, "3 new").

```jsx
<Badge tone="brand">Premium</Badge>
<Badge tone="accent">3 new</Badge>
<Badge tone="neutral" size="sm">Udvikling</Badge>
```

- `tone`: neutral · brand · success · warning · danger · info · accent.
- For live work status (approved/pending/overdue) prefer `StatusPill`, which includes a dot.
